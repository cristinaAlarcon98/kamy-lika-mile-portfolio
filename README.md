# Kamy Lika Mile Portfolio - Next.js & GSAP Project

This project is a personal portfolio developed for **Kamy Lika Mile**.  
It was built using **Next.js** and **GSAP** to create a smooth, modern, and animated portfolio experience.

## Description

This portfolio was inspired by the design and structure of my own portfolio:

It serves as a customized version adapted for Kamy Lika Mile, focusing on minimal design, elegant transitions, and clear presentation of content.  
The project combines **Next.js** for performance and routing with **GSAP** for animation and scroll effects.

## Tech Stack

- **Next.js**  
- **React**  
- **GSAP (GreenSock Animation Platform)**  
- **Tailwind CSS** for layout and styling  

## Installation

To run the project locally:

```bash
git clone https://github.com/cristinaAlarcon98/kamy-lika-mile-portfolio.git
cd kamy-lika-mile-portfolio
npm install
npm run dev
```

Then open your browser and visit:  
```
http://localhost:3000
```
